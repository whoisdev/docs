---

The Notebooks API gives you a single endpoint to chat with the best AI models from Anthropic, Google, OpenAI, and X-AI. Every request can use your notebook content as context, search the web, and run multi-step agent workflows.

<Note>
  **Base URL:** `https://dashboard.notebooks.app/api/v1`
</Note>

## Get started

<Steps>
  <Step title="Get your API key">
    Create an API key in your [dashboard](https://dashboard.notebooks.app/dashboard/settings/api). Keys start with `nb_sk_`.
  </Step>
  <Step title="Make your first request">
    Follow the [quickstart guide](/quickstart) to send a chat request and get a response in under 2 minutes.
  </Step>
  <Step title="Build something">
    Use the API to power chatbots, content tools, research assistants, or any AI-driven feature in your app.
  </Step>
</Steps>

## Key features

- **12 AI models** from Anthropic, Google, OpenAI, and X-AI through a single endpoint
- **Streaming** via Server-Sent Events for real-time responses
- **Agent mode** with multi-step reasoning and tool use
- **Web search** so the model can find up-to-date information
- **Structured output** to get JSON responses matching your schema
- **Notebook context** so the model knows about your content
